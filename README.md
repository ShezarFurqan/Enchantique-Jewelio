# Enchantique Jewelio

Enchantique Jewelio is a premium, full-stack e-commerce platform dedicated to jewelry and luxury items. Built using React, Tailwind CSS, and Express.js, it features a highly polished storefront for customers alongside a powerful, secure Admin Dashboard for seamless store operations and inventory management.

---

## ✨ Key Features

### 🛒 Customer Storefront
- **Elegant Product Catalog:** High-contrast, minimal UI showcasing premium products with detailed views.
- **Interactive Shopping Experience:** Fully functional cart system, dynamic pricing calculator, and responsive product filtering.
- **Seamless Checkout:** Structured checkout flow designed for maximum conversion rates and clean layout.

### 💼 Admin Management Dashboard
- **Product CRUD Operations:** Complete control to upload new products, edit existing listings, and delete inventory in real-time.
- **Image & Data Management:** Upload product images and manage descriptions, pricing, and stock status seamlessly.
- **Order Tracking:** Centralized dashboard to view, track, and manage incoming customer orders and shipping status.

---

## 🛠️ Tech Stack

### Frontend
- React.js (Component-driven single-page application architecture)
- Tailwind CSS (Utility-first, responsive, and professional layout design)

### Backend & Database
- Node.js & Express.js (Robust RESTful API endpoints for store and admin actions)
- Database Integration (Configured for dynamic product schema and order persistence)
